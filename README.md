# Progress Tracker

Full-stack progress tracking app built with Django and React.

## Purpose
This project documents my journey building a production-style application from scratch. 
The goal is not just features, but clean architecture, disciplined commits, and visible progress.

## Tech Stack
- Django (Backend API)
- Django REST Framework
- React + Vite (Frontend)
- Git & GitHub for iterative development

## Structure
backend/  - Django API
frontend/ - React client

## Current Status
Initial backend and frontend foundations complete.
